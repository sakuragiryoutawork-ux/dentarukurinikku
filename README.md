# デンタルクリニック(架空のコーポレートサイト)

## 概要

Web制作学習サイト「Web鍛」のコーディング課題を使用して制作しました。
1つ目の課題で学んだレスポンシブ対応やレイアウト調整を意識しながらコーディングを行いました。

## 使用技術

・HTML
・CSS


## 工夫した点


・スマートフォン版のデザインが用意されていなかったため、PCデザインを基に自己判断でレスポンシブ対応を実装しました。
ブレイクポイントを1つ設定し、画面幅に応じてレイアウトや余白を調整することで、スマートフォンでも見やすいデザインになるよう工夫しました。

・レスポンシブ対応時にヘッダー内のレイアウトを調整しました。
PC表示では display: flex; を使用してタイトルとキービジュアルを横並びに配置し、スマートフォン表示では縦並びになるようレイアウトを変更しています。

## 苦労した点

診療時間の表が、画面幅320px以下の場合に横にはみ出してしまう問題がありました。
レスポンシブ時のレイアウトを確認しながら余白や要素のサイズを調整し、画面幅が小さい場合でも表が崩れないように修正しました。
